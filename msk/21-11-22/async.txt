동기 통신과 비동기 통신
https://danac.tistory.com/251

동기식(Synchronous) 통신
    - 송수신측 간의 데이터를 주고받는 시점(시간적 관계)을 정확하게 일치(유지) 시키는 것 (정보통신기술용어해설)
    * 요청을 보내면 해당 요청이 처리되어 응답이 올 때까지 다음 동작을 진행하지 않고 대기, 항상 요청 A - 응답 A - 요청 B - 응답 B 형태로 통신이 진행됨
    * 요청과 응답을 '동기화' 한다고 생각하면 이해하기 좋을 듯

비동기식(Asynchronous) 통신
    - 송신, 수신 측 간에 시간 관계를 동시적으로 유지할 필요가 없는 전송 방식 (정보통신기술용어해설)
    * 요청을 보낸 뒤, 응답이 올 때까지 기다리지 않고 바로 다음 동작을 진행함
    * 요청 A - 요청 B - 요청 C - 응답 C - 응답 A - 요청 D - 응답 B - 응답 D - ...  형태로, 
    요청은 응답을 기다리지 않고 순차적으로 실행되며, 응답 또한 요청 순서와 무관하게 처리되는 순서대로 통신이 진행됨
