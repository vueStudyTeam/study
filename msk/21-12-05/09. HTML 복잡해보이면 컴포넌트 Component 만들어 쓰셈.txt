09. HTML 복잡해보이면 컴포넌트 Component 만들어 쓰셈

    - 축약해둔 컴포넌트 사용 방법
        1. vue 파일 import
        2. components 내에 원하는 이름으로 등록
            - import 해온 이름과 동일할 경우 "변수명"만 등록해도 됨(ES6)
        3. components 내에 등록된 컴포넌트명으로 사용

    - 컴포넌트 사용 이유
        1. 보기 좋음 (가독성)
        2. 재사용하기 쉬움

    - 주의사항
        - 반복적으로 출현할 부분만 컴포넌트화 권장
        - 컴포넌트 사용 시 코드가 복잡해질 수 있어 과도한 컴포넌트화 지양
