04. 리액트보다 100배 쉬운 Vue 반복문 v-for

    - 상단 Nav 메뉴 만들기

    - Vue의 HTML 반복문 (1) - 특정 횟수 반복
        - <태그 v-for="변수명 in 횟수" :key="변수명">
            - :key 속성 필수

    - Vue의 HTML 반복문 (2) - 특정 배열 반복
        - <태그 v-for="변수명 in 배열/객체" :key="변수명">
            - 배열/객체 내의 데이터 수만큼 반복
            - 변수에 데이터가 담김

    - :key의 용도
        - 반복문 돌린 요소를 컴퓨터가 구분하기 위해 사용
        - 반복문이 돌면서 변하는 숫자

    - Vue 반복문에서는 변수 2개까지 사용 가능
        - v-for="(변수명1, 변수명2) in 배열/객체"
        - 변수명1 : data, 변수명2 : index
