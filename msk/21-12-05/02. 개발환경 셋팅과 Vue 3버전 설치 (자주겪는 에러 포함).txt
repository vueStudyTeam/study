02. 개발환경 셋팅과 Vue 3버전 설치 (자주겪는 에러 포함)
    - npm install -g @vue/cli
    - 에디터 부가기능
        - Vetur
        - html css support
        - vue 3 snippets

    - vue create 프로젝트명

    - 미리보기 띄우고 싶으면 npm run serve

    - App.vue 파일이 메인페이지 내용
        - App.vue의 내용을 index.html에 html로 컴파일 (main.js)

    - node_modules : 프로젝트에 쓰는 라이브러리들
    - src : 소스코드 다 담는 곳
    - public : html파일, 기타파일 보관
    - package.json : 라이브러리 버전, 프로젝트 설명 기록
