06. v-if 와 모달창 만들기 (Vue에서 동적인 UI 만드는 법)

    - 이미지 넣기
        - 절대경로 : 그냥 입력
        - 상대경로 : src/assets 폴더 아래에 지정한 경우 "./assets/이미지경로" 형식으로 사용 (현재 경로가 ./)
        - Webpack 문법상 무조건 "./"로 시작해야 함

        ** 이미지 src를 데이터바인딩하여 넣을 때
            - 단순 String을 데이터로부터 받아올 경우 정상적으로 경로를 읽어들이지 못함
            - 경로를 import 한다는 의미로 require 함수를 사용하여 경로를 읽어들인 후 데이터바인딩 필요
            - 출처 : https://m.blog.naver.com/lizziechung/221793761299

    - 모달 만들기 (동적인 UI 만드는 법)
        0. HTML CSS 디자인
        1. UI의 현재 상태를 데이터로 저장
        2. 데이터에 따라 UI가 어떻게 보일지 작성 (v-if 사용)
