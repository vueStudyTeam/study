08. props 싫으면 slot은 어떻습니까

    - slot으로 부모-> 자식 데이터 전송하기
        1) 자식 컴포넌트에 <slot></slot>으로 데이터 받는 자리 만들기
        2) 부모 컴포넌트에서 <자식 컴포넌트>slot에 들어갈 데이터</자식 컴포넌트> 형태로 자식 컴포넌트 작성

    - slot 사용 시 주의사항
        1) 태그 안에 데이터바인딩 할 때만 slot 사용 가능
        2) 여러 개의 slot을 사용할 때
            - 자식 컴포넌트 : <slot name="슬롯명"></slot>
            - 부모 컴포넌트 : <자식 컴포넌트><template v-slot="데이터"></template></자식 컴포넌트>
                - 여러 개의 slot이 필요할 때는 차라리 props로 보내는 게 나을 수도.

    - slot 태그 장점
        - 간단한 데이터 전송에 유리함
        - 자식 컴포넌트에 HTML을 그대로 전송할 수도 있음
        - 부모 컴포넌트가 자식 컴포넌트의 데이터를 사용할 때 slot 사용 가능
            - 자식 컴포넌트 : <slot :자식변수명="데이터"></slot>
            - 부모 컴포넌트 : <자식 컴포넌트> <template v-slot:default="변수명">{{변수명.자식변수명}}</template> </자식 컴포넌트>
